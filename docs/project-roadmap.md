# Project Roadmap

Last audited: 2026-06-01

This document tracks what still needs to be done across product, public content, admin workflow, research operations, and documentation. It is meant to sit above the generated research queue and below the product brief.

## Current State

The project is a working alpha of the Hallmarks of Aging LEV Tracker.

- The public app builds and has routes for overview, hallmarks, tracks, interventions, studies, findings, activity, methods, state-of-the-field notes, and admin review.
- The homepage implements the current wireframe direction: overall outlook, hallmark grid, recent changes, 2036 scenario lens, and trust notes.
- The file-backed admin review workflow exists for candidate bundles, review comments, evidence reviews, promotion readiness, and publication events.
- The research planning layer is active: `research/state/coverage-status.v1.json` and `research/backlog/track-priority.v1.json` are regenerated by `npm run sync:research-planning`.
- Track coverage is still early, but every hallmark group now has at least one anchor baseline: 22 of 38 seeded tracks have published baseline coverage and 16 tracks do not yet have a published baseline.
